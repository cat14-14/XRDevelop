# F-16 Public Reference Dossier

Last updated: 2026-04-12

## Purpose

This file is a public-source F-16 reference package for this project.

It is not literally "all information" about the F-16, because no single public document can contain every open-source detail, block difference, retrofit state, national customization, or weapon certification history. Instead, this file consolidates the highest-value public facts that are useful for:

- simulation baselining
- asset validation
- tuning anchors
- variant separation
- operator and program context
- traceable citation of public sources

## 1. Aircraft Identity

- Official name: `F-16 Fighting Falcon`
- Common pilot nickname: `Viper`
- Original manufacturer: `General Dynamics`
- Current prime contractor: `Lockheed Martin`
- Role: single-engine multirole fighter
- Core design intent: lightweight day fighter that evolved into a multirole strike / air-defense platform

Lockheed Martin's historical material and anniversary coverage note that the YF-16 first flew unintentionally during a high-speed taxi event on `1974-01-20`, followed by the official first flight on `1974-02-02`.

The U.S. Air Force fact sheet describes the F-16 as a compact multirole fighter with strong maneuverability, high combat effectiveness, and broad mission flexibility across air-to-air and air-to-surface roles.

## 2. High-Level Timeline

- `1974-01-20`: unplanned first YF-16 takeoff during taxi test
- `1974-02-02`: official first flight of YF-16
- `1979-01`: F-16A initial operating capability per USAF fact sheet
- `1981`: F-16C/D Block 25-32 initial operating capability per USAF fact sheet
- `1989`: F-16C/D Block 40-42 initial operating capability per USAF fact sheet
- `1994`: F-16C/D Block 50-52 initial operating capability per USAF fact sheet
- `2003`: first flight of F-16E/F Block 60 per Lockheed Martin
- `2012`: F-16V announced by Lockheed Martin
- `2025-05`: Lockheed Martin fast-facts sheet states data current as of May 2025

## 3. Core Public Characteristics

### Airframe / handling identity

- relaxed static stability
- fly-by-wire flight controls
- bubble canopy / strong outward visibility
- side-stick controller concept
- blended wing-body shaping
- high agility with strong energy maneuvering performance

Lockheed Martin's 30th-anniversary history explicitly highlights these design cues:

- fly-by-wire flight control
- relaxed static stability
- wing-body blending
- variable camber wing
- high-g cockpit
- integrated digital avionics using MIL-STD-1553 data bus
- modular construction

## 4. Baseline Physical Data

These values vary by block and configuration. Use them as public anchors, not as a complete truth set for every F-16.

### F-16C/D public fact-sheet anchor

Source basis: USAF / ANG fact-sheet mirrors.

- Length: `49 ft 5 in` / `14.8 m`
- Wingspan: `32 ft 8 in` / `9.8 m`
- Height: `16 ft` / `4.8 m`
- Empty weight: `19,700 lb` / `8,936 kg`
- Maximum takeoff weight: `37,500 lb` / `16,875 kg`
- Internal fuel: `7,000 lb` / `3,175 kg`
- Typical fuel with two external tanks: `12,000 lb` / `5,443 kg`
- Speed: `1,500 mph` / `Mach 2 at altitude`
- Ferry range: `more than 2,002 mi` / `1,740 nmi`
- Ceiling: `above 50,000 ft` / `15 km`
- Design g limit with full internal fuel: `9 g`
- Crew: `F-16C = 1`, `F-16D = 1 or 2`

### F-16 Block 70/72 public product-card anchor

Source basis: Lockheed Martin January 2025 product card.

- Length: `49.3 ft` / `15.027 m`
- Wingspan: `31 ft` / `9.449 m`
- Height: `16.7 ft` / `5.09 m`
- Empty weight: `20,300 lb` / `9,207 kg`
- Engine thrust class: `29,000 lb` / `13,000 kg`
- Maximum takeoff gross weight: `48,000 lb` / `21,772 kg`
- Speed: `1,500 mph` / `Mach 2+`
- Design load factor: `9 g`
- Structural service life: `12,000 hr`

## 5. Engines

Public official references show that engine fit depends on block and customer.

### Legacy C/D family

Official fact-sheet mirrors list:

- Pratt & Whitney `F100-PW-200/220/229`
- General Electric `F110-GE-100/129`

Public thrust anchor for F-16C/D:

- `27,000 lb` thrust class

### Block 60 / F-16E/F

Lockheed Martin states the Block 60 / F-16E/F uses:

- `F110-GE-132`
- about `32,500 lb` thrust

### Block 70/72

Lockheed Martin product material gives:

- `29,000 lb` thrust class
- depending on customer, aircraft can be aligned with either GE or Pratt engine families

## 6. Avionics, Sensors, and Mission Systems

### Legacy F-16 evolution

Public sources show continuous upgrades from A/B to C/D, then Block 60, then F-16V / Block 70/72.

### F-16V / Block 70/72 official public features

Lockheed Martin lists the following as headline features:

- `APG-83 AESA radar`
- upgraded mission computer and avionics architecture
- new cockpit displays
- improved radar modes
- IRST integration
- advanced data link
- targeting pod integration
- advanced weapons support
- digital flight control computer
- enhanced autopilot / auto-throttle
- `Auto GCAS`
- digital intercom with 3D audio
- precision GPS / INS navigation

Lockheed Martin also states:

- the APG-83 gives the F-16 "5th generation fighter radar capabilities"
- Block 70/72 uses a new center pedestal display for high-resolution tactical imagery
- the APG-83 shares `95% software commonality` and `70% hardware commonality` with the F-35 radar
- structural life is extended to `12,000 hours`
- the platform is intended to remain relevant into the `2060s and beyond`

### Block 60 official public features

Lockheed Martin's public Block 60 material identifies:

- `APG-80` AESA radar
- internal FLIR navigation / targeting system
- advanced cockpit with large color displays
- fiber-optic avionics data bus
- Falcon Edge internal electronic countermeasures system
- conformal fuel tank compatibility

## 7. Weapons / Stores / Mission Flexibility

### USAF public baseline

The USAF fact sheet lists:

- one `M-61A1 20 mm` cannon with `500` rounds
- external stations able to carry up to six air-to-air missiles
- conventional air-to-air and air-to-surface munitions
- electronic countermeasure pods

Example payload listed in the fact sheet:

- two `2,000 lb` bombs
- two `AIM-9`
- two `AIM-120`
- two `2,400 lb` external fuel tanks

### Lockheed Martin public integration claim

Lockheed Martin states:

- more than `3,300` carriage and release configurations have been certified
- for more than `180` weapon and store types

That makes the F-16 one of the most broadly integrated multirole fighter platforms in open official literature.

## 8. Flight-Control and Handling Cues Relevant to Simulation

The NASA `AFTI/F-16 Digital Flight Control System Experience` paper is especially useful for simulation design because it is public and directly discusses control-law behavior.

### Public cues from NASA

- the AFTI/F-16 investigated a `triplex`, `dual fail/operate` digital flight control system
- the modified test aircraft added canards and a dorsal fairing for advanced control experiments
- the digital system included mission-specific and decoupled modes
- mode selection affected both avionics and flight control
- pitch-axis behavior in some advanced modes used pitch-rate-error-based adaptation
- decoupled options included pointing, translation, and direct-force modes
- the system retained an `analog reversion mode` as backup from any digital mode

### Practical simulation takeaway for this repo

For a public-data baseline, the safest first-order F-16 control-law interpretation remains:

- pitch: `Nz-command-like` behavior with AoA protection
- roll: `roll-rate-command-like` behavior
- yaw: stability augmentation / departure resistance oriented
- landing / approach: more pitch-rate-like feel is publicly defensible

Those conclusions are consistent with the existing implementation-plan assumptions already in this repo.

## 9. Variants Summary

### F-16A/B

- early production single-seat / two-seat versions
- original lightweight-fighter lineage

### F-16C/D

- broader multirole standard
- common public spec anchor used in USAF fact sheets
- major block families include `25/32`, `40/42`, `50/52`

### F-16E/F Block 60

- official Lockheed Martin designation for the advanced UAE-oriented generation
- stronger engine, deeper avionics changes, major cockpit refresh, AESA radar

### F-16V

- official Lockheed Martin modernization path announced in 2012
- introduces AESA radar, mission computer and cockpit upgrades
- available both as upgrade path and configuration basis for new-build jets

### F-16 Block 70/72

- current flagship production form in official Lockheed Martin material
- described by Lockheed Martin as the newest and most advanced production F-16

## 11. Operational Use

The USAF fact sheet says the F-16 has been a major component of U.S. combat forces after September 11, 2001, flying thousands of sorties in support of:

- `Operation Noble Eagle`
- `Operation Enduring Freedom`
- `Operation Iraqi Freedom`

In public NATO and allied service, the F-16 remains important because it combines:

- mature sustainment
- large global user base
- wide weapon compatibility
- interoperability with U.S. and NATO systems
- meaningful upgrade headroom through radar, cockpit, EW, and software modernization

## 12. Known Public Differences Between Legacy and New-Build F-16s

Do not collapse all F-16 data into one model.

Important public differences include:

- dimensions vary slightly by variant and installed configuration
- mass varies significantly across blocks
- thrust class varies across engine / block combinations
- avionics architecture differs radically between legacy, Block 60, F-16V, and Block 70/72
- sensor fit differs by nation and retrofit level
- weapon certification differs by customer
- service-life assumptions differ between earlier production aircraft and Block 70/72

For this repo, that means a single hardcoded "F-16" data model will eventually become a limitation.

## 13. Recommended Sim Baseline For This Project

If the goal is a defensible public-data baseline, the most stable starting point is:

- geometry anchored to public `F-16C/D` values
- control-law direction informed by NASA AFTI/F-16 public papers
- engine tuned to the `27,000 lb` public C/D thrust-class anchor unless the sim explicitly targets Block 70/72
- structural / performance targets bounded by:
  - `Mach 2 class`
  - `50,000+ ft ceiling`
  - `9 g`
  - `7,000 lb internal fuel`

If the goal is a modernized export / latest-production representation, then use a separate configuration that tracks:

- `Block 70/72`
- `APG-83 AESA`
- `12,000 hr service life`
- `29,000 lb thrust class`
- higher gross weight anchor

## 14. What Public Sources Do Not Fully Give You

Even with extensive public references, the following are still incomplete or highly configuration-specific:

- full aerodynamic coefficient tables across Mach / AoA / beta
- exact flight-control gain schedules for frontline variants
- complete EM charts for every block and store configuration
- detailed engine deck data by altitude / Mach / throttle state
- nation-specific radar / EW parameter sets
- full weapon release envelopes and software tapes
- exact drag increments for every store combination

Those gaps are normal. A usable sim can still be built, but it must treat these areas as tuned approximations rather than directly known truth.
